# 🎓 AI Study Help

An AI-powered study assistant that helps students learn smarter by generating summaries, quizzes, flashcards, and answering questions from study materials.

🌐 **Live Demo:** https://ai-study-help.netlify.app/

---

## 📌 Features

- 🤖 AI Study Chatbot
- 📝 AI Note Summarizer
- 🧠 Flashcard Generator
- ❓ Quiz Generator
- 📄 PDF Upload & Processing
- 🎤 Voice-to-Text Input
- 🌙 Dark Mode
- 📱 Fully Responsive Design

---

## 🚀 Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript

**AI & APIs**
- Google Gemini API *(or your AI provider)*
- PDF Processing API

**Deployment**
- Netlify

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/parthabit/ai-study-help.git
```

Go to the project folder

```bash
cd ai-study-help
```

Open

```text
index.html
```

or use Live Server in VS Code.

---

## 📖 How It Works

1. Enter your notes or upload a PDF.
2. Select the desired AI tool.
3. AI processes the content.
4. Receive summaries, quizzes, flashcards, or answers instantly.
